# 🗳️ @inseme/brique-democracy - Shared Democracy Specifications

**@inseme/brique-democracy** is a technical package containing shared specifications and tests for democratic processes within the Inseme monorepo.

> For an overview of the ecosystem (Agora, IA, Multi-instances), see the
> [global repository README](../../README.md).

---

## 🎯 What is it for?

This package serves as a central point for testing and validating the core democratic logic used across various modules (like `brique-kudocracy`). It ensures that the principles of liquid democracy and voting are consistently applied and verified.

---

## 🛠️ Project Structure

```
packages/brique-democracy/
└── tests/             # Specification tests for core democracy logic
```

## ⚖️ Neutrality & Commitment
